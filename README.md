## 今天做点什么？

每天最多3件。

**早上：**
- 喝口水，感受温度
- 看窗外30秒，别看手机
- 对镜子笑一下（挺傻的但是有用）

**下午：**
- 站起来伸个懒腰
- 摸一下身边的植物
- 发会儿呆

**晚上：**
- 关灯，坐着什么都不干
- 想想今天哪一刻还不错
- 早点睡，少刷会儿手机
